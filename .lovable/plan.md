# Prathamesh Gite — Data Analyst Portfolio

## Overview

A premium, recruiter-focused portfolio website with a dashboard-style aesthetic that emphasizes business insights and data storytelling rather than developer skills.

## Pages & Sections

### 1. Hero Section

- Name, title ("Data Analyst"), tagline "Turning Data into Actionable Insights"
- Location badge (Kalyan, India)
- KPI cards row: **4 Projects** | **8+ Tools** | **1+ Year Experience**
- CTA buttons: "Download Resume" + "Contact Me"
- Subtle animated data-grid/chart pattern in background

### 2. About Section

- Clean two-column layout: bio text + a professional stats/highlights card
- Emphasis on data cleaning, EDA, dashboards, and decision-making

### 3. Skills Section

- Organized in categorized card groups (Programming, Visualization, Databases, Tools, Concepts)
- Progress-bar or badge style per skill — clean, not flashy
- Dashboard-style layout with subtle borders and icons

### 4. Projects Section (Case Study Style)

- Each project as a case-study card with:
  - Title, status badge (Completed / In Progress)
  - Key insight bullets (business-focused, not tool-focused)
  - KPI highlights (e.g., "8,800+ records analyzed")
  - Tags for tools used
- Four projects: Netflix, Titanic, Customer Complaints, Fraud Detection

### 5. Experience Section

- Timeline layout with role, company (Lionbridge Technologies), and description
- Education entry: Bsc in IT (2023)

### 6. Contact Section

- Simple contact form (Name, Email, Message) with validation
- GitHub and LinkedIn icon links
- Location info

## Global Features

- **Sticky navbar** with smooth scroll: Home, About, Skills, Projects, Experience, Contact
- **Light/Dark mode toggle** in navbar
- **Fully responsive** (mobile hamburger menu)
- **Resume download button** in hero and navbar
- **Subtle animations** on scroll (fade-in, slide-up via CSS)
- Dashboard-inspired color palette: clean whites/grays in light mode, deep navy/slate in dark mode
- Analytics-style decorative elements (subtle grid lines, chart motifs)
